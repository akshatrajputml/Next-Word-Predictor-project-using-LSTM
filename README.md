# 🤖 Next Word Predictor using LSTM

A Deep Learning NLP project that predicts the next most probable word based on user input text using an LSTM (Long Short-Term Memory) neural network.

---

## 📌 Project Overview

This project applies Natural Language Processing (NLP) and Deep Learning techniques to generate next-word predictions from text sequences.

The application takes user text input and predicts the most likely next word through a trained LSTM model.

---

## 🚀 Features

- Text preprocessing and tokenization
- Sequence generation
- LSTM-based prediction model
- Real-time prediction using Gradio UI
- Interactive deployment interface
